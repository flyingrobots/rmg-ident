# Security Policy

## Supported Versions

We are committed to providing security updates for the latest version of this crate.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it to us by sending an email to james@flyingrobots.dev. We will work with you to understand and resolve the issue.
