# Contributing

We welcome contributions to this project!

## Getting Started

- Fork the repository on GitHub.
- Clone your fork locally.
- Create a new branch for your feature or bug fix.

## Making Changes

- Make your changes, ensuring that you follow the existing code style.
- Add or update tests as appropriate.
- Ensure that all tests pass by running `cargo test`.
- Ensure that your code is well-formatted by running `cargo fmt`.
- Ensure that your code is free of common mistakes by running `cargo clippy`.

## Submitting Your Changes

- Push your changes to your fork on GitHub.
- Create a pull request to the `main` branch of the original repository.
- In your pull request, describe the changes you have made and why.

Thank you for your contribution!
